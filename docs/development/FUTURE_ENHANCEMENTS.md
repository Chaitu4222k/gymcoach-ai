## Athletic Performance Mode

Description

Allow users to train for a specific sport instead of a general fitness goal.

Possible Sports

* Cricket
* Football
* Basketball
* Tennis
* Badminton
* Volleyball
* Running
* Martial Arts

Potential AI Enhancements

* Sport-specific workout plans
* Sport-specific mobility routines
* Injury prevention exercises
* Performance-focused recommendations

Status

Planned
